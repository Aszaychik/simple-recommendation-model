Menu Recommendation API Documentation

Use the API endpoint /api/recommend to get menu recommendations for a customer.

Send a POST request with JSON payload: {"customer_id": 1}

Example Response:

{
    "customer_id": 1,
    "recommendations": [
        {"item_id": 5, "predicted_rating": 8.2},
        {"item_id": 3, "predicted_rating": 7.9},
        {"item_id": 7, "predicted_rating": 7.5},
        {"item_id": 2, "predicted_rating": 7.1},
        {"item_id": 10, "predicted_rating": 6.8}
    ]
}
